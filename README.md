## 核心功能跑一遍

**一键存档 / 恢复**
操作很直接，不需要配置什么复杂参数。指定一个文件夹，什么时候存档你说了算，恢复到任意一个历史状态只需要点一下。

**增量存储，不吃硬盘**
这个技术细节值得一提：每次存档只保存"有变化的那部分"，不是把整个文件夹复制一遍。一个 100MB 的文件，你改了 10KB，存档只占 10KB 的额外空间。如果你是手动拷贝文件夹来管版本的，换算下来能省好几倍空间。

**版本差异对比**
可以直接对比两个存档之间的文件差异，哪些文件新增了、哪些被删了、哪些被修改过，一目了然，还支持调用第三方工具做文件内容级别的对比。

**迭代关系图**
这个功能稍微有点意思——存档之间的关系用类似思维导图的形式展示，分支路径都看得清楚。如果你的文件在多个方向上改了很多版，这个视图能帮你搞清楚"这个版本是从哪里来的"。

**时间轴视图**
按时间线排列存档，密度高，适合快速翻找。

**Mini 模式**
可以缩成一个悬浮小窗，甚至缩成一个点，但照样能标识文件是否有变更。不占屏幕，日常用起来挺舒服的。

**搜索存档说明**
如果你每次存档都写了备注，这个搜索功能就会变得非常实用——直接搜关键词就能定位目标版本，不用一条条翻时间线。
